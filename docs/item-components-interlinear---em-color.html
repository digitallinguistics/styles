2.9.1 --em-color

Set the color of emphasized text by giving a value to the --em-color variable.

Example

.igl { --em-color: red; }

siksink his heːčtiʔi

  1. siksi‑nk eagle‑??
  2. his resp
  3. heːčt‑iʔi call‑nf;sg

an eagle met him
