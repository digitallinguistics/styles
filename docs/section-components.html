5.1.3 Pre

Used for standalone code blocks (for inline code blocks, see the <code> tag). If the <pre> element is used by itself, without a <code> element inside it, it will simply display as preformatted text. If a <code> element is included inside the <pre>, however, that element will be display: block;.

Example

This is a <pre> element without a <code> element inside of it.
  This is a <pre> element with a <code> element inside of it.

It is displayed as a standalone block of code.

{

  "dateModified": "2017-03-13T04:51:26.134Z",

    "eng": "Chitimacha",
    "ctm": "Sitimaxa"
  }
}

Source: lists/bulleted/bulleted.less, line 1

5.6.1 Bulleted List

In DLx projects, lists are often used semantically rather than stylistically, e.g. a phrase includes a list of words. As such, lists have no styling by default, and list styling needs to be explicitly set with the .bulleted or .numbered styles. List items with text should include a <p> tag or other relevant element within them, following the principles of semantic markup.

Example

This is text leading up to the bulleted list (for reference).

  • This is the first list item.
  • This is the second list item.
  • This is the third list item.
<p>This is text leading up to the bulleted list (for reference).</p>
<ul class=bulleted>
  <li>This is the first list item.</li>
  <li>This is the second list item.</li>
  <li>This is the third list item.</li>
</ul>

Source: lists/numbered/numbered.less, line 1

5.6.3 Reference List

Styling for a bibliographic reference list. Note that each reference should be an <li> tag containing a <p> tag. NOTE: You may have to further customize the styling of the <hX> element depending on which header you are using. Use <cite> for publication titles.

Example

Reference List

This is a paragraph giving a short description of this reference list.

  1. Hall, Patrick J. 2014. Serialization and hierarchy: From data to corpus in linguistic fieldwork. M.A. thesis. Linguistics, University of California, Santa Barbara.

  2. Hieber, Daniel W. 2015. Review: Endangered languages & new technologies (Jones). Language Documentation & Conservation 9: 344350.

<section class=reference-list>

  <h2>Reference List</h2>

  <p>This is a paragraph giving a short description of this reference list.</p>

  <ol>

    <li>
      <p>Hall, Patrick J. 2014. <cite>Serialization and hierarchy: From data to corpus in linguistic fieldwork</cite>. M.A. thesis. Linguistics, University of California, Santa Barbara.</p>
    </li>

    <li>
      <p>Hieber, Daniel W. 2015. Review: Endangered languages &amp; new technologies (Jones). <cite>Language Documentation &amp; Conservation</cite> 9: 344350.</p>
    </li>

  </ol>

</section>
